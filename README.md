# Daily Quotes PWA - Retro Edition

A **pixelated retro** Progressive Web App for viewing and managing daily quotes with a classic 80s/90s DOS aesthetic!

## 🎮 Features

✨ **Progressive Web App (PWA)**
- Install on home screen (mobile & desktop)
- Works offline with Service Worker
- Fast loading with caching

💬 **Quote Management**
- View random quotes
- Add your own quotes
- Local storage persistence

📱 **Retro Design**
- Authentic DOS/CRT monitor styling
- Pixelated fonts (Press Start 2P)
- CRT scanlines effect
- 8-bit color scheme

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

## 🌐 Deploy to Vercel

```bash
npm install -g vercel
vercel
```
